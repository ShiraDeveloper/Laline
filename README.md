# LALINE

## Overview
LALINE is a static design website built using HTML and CSS.  
The project focuses on UI/UX design, layout structure, and responsive styling.

## Features
- Clean and modern UI design
- Responsive layout for different screen sizes
- Structured HTML pages
- Styled using CSS

## Technologies
- HTML5
- CSS3

## Purpose
This project was created to practice front-end design skills and improve understanding of layout building and responsive web design.

## Author
Developed as a personal front-end design project.
